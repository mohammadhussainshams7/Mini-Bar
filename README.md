# Mini-Bar

A web application built with Laravel and Blade for managing a mini bar system. This project demonstrates CRUD operations, user authentication, and a simple inventory system.

## Features

- User registration and login
- Product and category management
- Inventory tracking
- Responsive UI using Blade templates
- RESTful API endpoints

## Technologies Used

- PHP 8.x
- Laravel 10
- Blade Templating
- MySQL / SQLite
- CSS / JavaScript
- Composer & NPM for dependency management
